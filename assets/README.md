# assets/README.md
# This folder holds Cindy's avatar image(s).
#
# Place a file called `cindy_avatar.png` (or `.jpg`) here and the app will
# display it as Cindy's profile picture.  If no image is found the app
# automatically falls back to the 🤖 emoji placeholder — so the app runs
# perfectly even without any image files committed to the repository.
#
# Recommended size: 256 × 256 px, transparent background (PNG).
# Keep images age-appropriate and consistent with Cindy's cheerful robot theme.
